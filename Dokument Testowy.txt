Wzane: 

Pamietaj ze aby commitowac zmiany wprowadzone w plikach musza one one byc dodane do repo za pomoca komendy git add . 

Nastepnie uzywasz komendy git commit -m "". Sprawdzasz za pomoca komendy git status czy towje zmianay zostaly zacommitowane i czy pracujesz na czystym pbranchu 

Jak upewnisz sie ze zmiany zostaly dodane i zacommitowane uzywasz komendy git push i wypychasz zmiany do zdlanego repo na git hubie.
